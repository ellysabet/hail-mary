# 🚀 프로젝트 헤일메리 - 우주산업 진로탐험

중학생 대상 우주산업 진로교육 프로그램

## 📚 프로그램 소개

프로젝트 헤일메리는 중학생들이 우주산업 관련 직업을 체험하고, 팀 활동을 통해 협력과 문제해결 능력을 기르는 인터랙티브 교육 게임입니다.

## 🎮 게임 구성 (6개 라운드)

### Round 1: 새로운 행성 발견 🪐
- **직업**: 우주생물학자
- **미션**: 카드 매칭 게임으로 생명체 필수 요소 찾기
- **학습 목표**: 생명체 거주 가능 조건 이해

### Round 2: 로켓 발사 🚀
- **직업**: 항공우주공학자
- **미션**: 종이비행기 발사 (오프라인 활동)
- **학습 목표**: 작용-반작용의 법칙

### Round 3: 태양전지판 ⚡
- **직업**: 우주 에너지 엔지니어
- **미션**: 3가지 접기 방식 체험 및 각도 조절
- **학습 목표**: Z-Fold, Miura-Fold, Accordion-Fold

### Round 4: 외계 행성 탐사 로봇 🤖
- **직업**: 우주 로봇공학자
- **미션**: 5단계 순차 탐사 (화살표 키 조작)
- **학습 목표**: 로봇 탐사 과정 이해

### Round 5: 지구 귀환 🌍
- **직업**: 궤도 역학 전문가
- **미션**: 2단계 재진입 (속도/각도 조절)
- **학습 목표**: 우주속도, 자유귀환궤도

### Round 6: 지속가능한 우주 ♻️
- **직업**: 우주 환경 전문가
- **미션**: 팀 토론 + Canvas 그림판으로 포스터 제작
- **학습 목표**: 우주 쓰레기 문제 인식 및 해결방안 제시

## 🛠️ 기술 스택

- **Frontend**: React + Vite
- **Language**: JavaScript
- **Styling**: CSS
- **Storage**: localStorage (세션 관리)

## 🚀 설치 및 실행

```bash
# 1. 저장소 클론
git clone https://github.com/ellysabet/hail-mary.git

# 2. 폴더 이동
cd hail-mary

# 3. 패키지 설치
npm install

# 4. 개발 서버 실행
npm run dev
```

## 👥 사용 방법

1. **선생님**: 세션 생성 및 라운드 진행
2. **학생**: 세션 코드 입력 후 팀별로 참여
3. 각 라운드마다 직업 소개 → 미션 수행 → 퀴즈 풀이
4. 팀별 점수 집계 및 최종 순위 발표

## 🎯 교육 효과

- 우주산업 관련 직업 체험
- 팀워크 및 협업 능력 향상
- 과학 원리 실습을 통한 학습
- 창의적 문제 해결 능력 배양
