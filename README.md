# 🚀 프로젝트 헤일메리 - 우주산업 진로탐험

> 중학생 대상 우주산업 진로교육 인터랙티브 게임

**배포 주소**: https://ellysabet.github.io/hail-mary/

---

## 📚 프로그램 소개

프로젝트 헤일메리는 중학생들이 우주산업 관련 직업을 체험하고, 팀 활동을 통해 협력과 문제해결 능력을 기르는 실시간 인터랙티브 교육 게임입니다.

교사가 대시보드에서 라운드를 제어하고, 학생들은 세션 코드로 입장하여 팀을 구성한 뒤 6개의 라운드를 함께 진행합니다.

---

## 🎮 게임 구성 (6개 라운드)

### Round 1: 새로운 행성 발견 🪐
- **직업**: 우주생물학자
- **미션**: 카드 매칭 게임으로 생명체 필수 요소(물, 햇빛, 온도, 대기) 찾기
- **학습 목표**: 생명체 거주 가능 조건 이해

### Round 2: 로켓 발사 🚀
- **직업**: 로켓공학자
- **미션**: A4 용지로 종이 로켓 제작 후 거리/정확도 도전 (오프라인 활동)
- **학습 목표**: 무게중심, 공기역학, 작용-반작용의 법칙

### Round 3: 태양전지판 ⚡
- **직업**: 우주 에너지 엔지니어
- **미션**: Z-Fold / Miura-Fold / Accordion-Fold 3가지 접기 방식 체험 및 태양 각도 맞추기
- **학습 목표**: 태양전지판 설계 원리, 에너지 효율

### Round 4: 외계 행성 탐사 로봇 🤖
- **직업**: 우주 로봇공학자
- **미션**: 탐사 로봇을 조종해 에너지 확인 → 온도 측정 → 물 탐지 → 토양 샘플 → 사진 촬영의 5단계 순차 탐사
- **학습 목표**: 로봇 탐사 과정, 순차적 문제 해결

### Round 5: 지구 귀환 🌍
- **직업**: 궤도 역학 전문가
- **미션**: 속도와 각도를 조절해 대기권 재진입 성공
- **학습 목표**: 우주속도, 자유귀환궤도, 재진입 열차폐

### Round 6: 지속가능한 우주 ♻️
- **직업**: 우주 환경 전문가
- **미션**: 팀 토론 후 Canvas 그림판으로 우주 쓰레기 해결 아이디어 포스터 제작
- **학습 목표**: 우주 쓰레기 문제 인식 및 해결방안 제시

## 🏗️ 주요 기능

### 교사 대시보드
- 비밀번호 인증 후 세션 생성
- 기존 진행 중인 세션 재입장 가능
- 라운드별 단계 제어 (직업설명 완료 / 미션 시작 / 퀴즈 시작)
- **강제 단계 이동** - 뒤처진 학생을 현재 단계로 즉시 이동
- 실시간 팀 현황 및 점수 확인
- Round 6 팀 포스터 갤러리 (클릭 시 확대)
- 팀 순위 / 개인 순위 탭 (언제든 확인 가능)
- 수업 종료 시 세션 완전 삭제

### 학생 화면
- 세션 코드 입력 후 팀 선택 또는 생성
- 팀 재입장 시 유령 멤버 방지 (중복 이름 처리)
- 실시간 라운드 진행 상태 동기화
- 최종 결과 시 행성 탐사 성공 축하 엔딩 시퀀스 (8장면, ~15초)
- 수업 종료 시 재접근 완전 차단

### 점수 체계
- 라운드별 퀴즈 정답 시 **팀 100점 + 개인 100점** 동시 부여
- 개인 만점: **600점** (6라운드 × 100점)
- 팀 만점: **2,400점** (4명 기준)
- 개인 등급: 우주 훈련생 → 우주 대원 → 우주 탐험가 → 우주 전문가 → 우주 마스터

## 👥 사용 방법

### 교사
1. 메인 화면에서 **교사 - 수업 시작하기** 클릭
2. 교사 암호 입력 후 세션 생성 (또는 기존 세션 재입장)
3. 학생들에게 세션 코드 공유
4. 라운드별 버튼으로 수업 진행 제어
5. 모든 라운드 완료 후 **최종 결과 보기** → **수업 종료**

### 학생
1. 메인 화면에서 **학생 - 미션 참여하기** 클릭
2. 세션 코드 입력 → 이름 입력 → 팀 선택 또는 생성
3. 선생님의 라운드 시작 신호를 기다린 후 미션 수행
4. 각 라운드 퀴즈 완료 후 다음 라운드 대기

### 테스트 팁
- 교사/학생 동시 테스트 시 **다른 브라우저** 또는 **시크릿 모드** 사용 권장
  - 같은 브라우저는 localStorage를 공유하여 간섭 발생
